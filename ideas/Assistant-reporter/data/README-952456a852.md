# 技术趋势监测脚本

## 概述
本脚本用于定期扫描技术趋势相关的信源，提取大模型、Agent Skill、OpenSpec等技术前沿内容，为周度深度简报提供数据支持。

## 功能特性
- 支持多信源扫描（GitHub Trending、Coze博客等）
- 基于关键词的内容筛选
- 自动去重（基于内容哈希）
- 内容分类（技术突破、工具更新、行业动态、趋势研判）
- 结果保存为结构化JSON文件
- 生成监测报告

## 配置文件
信源配置位于 `../data/信源配置.json`，包含：
- 公众号列表（需特殊处理）
- 社区信源
- 搜索关键词

## 使用方法

### 1. 安装依赖
```bash
pip install -r requirements.txt
```

### 2. 运行监测
```bash
python monitor.py
```

### 3. 输出文件
- 原始数据：`../data/raw/raw_YYYYMMDD.json`
- 监测报告：`../outputs/周报/监测报告_YYYYMMDD.md`

## 信源扩展
要添加新信源，需修改：
1. `process_source()` 函数添加新的信源类型处理逻辑
2. 在主函数中添加相应的信源配置

## 注意事项
- 微信公众号内容需通过微信开放平台API或RSS服务获取
- GitHub Trending页面结构可能变化，需定期更新解析逻辑
- 建议设置定时任务（如cron）每周自动运行
