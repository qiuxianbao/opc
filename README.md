### 关键名词

F值-》光圈 -》景深

```
𝑓 值(𝑁)= 𝑓 / 𝐷

f = 镜头焦距（Focal Length）
平行光线穿过透镜后，聚焦到焦点（传感器）的距离

D = 光圈的有效孔径（Entrance Pupil Diameter）
镜头内部光圈叶片张开的物理直径大小。它决定了有多少光线能够进入镜头
```
