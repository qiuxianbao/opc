# 下载经典台词 (Download Classic Quotes)

## 技能定位
本技能用于识别用户意图，当用户要求下载某部电影或某个作者的"经典台词"时，自动调用此技能将台词整理并保存到 `/root/data/` 目录下。

**核心能力：**
1. 意图识别：自动识别用户要求下载经典台词的意图
2. 信息提取：提取作品名、作者名/导演名
3. 智能补充：如果只有作品名，自动搜索导演名
4. 情感分类：按励志/爱情/幽默/哲理/经典等情感维度分类
5. 文件保存：按标准格式保存到 `/root/data/{作者名}_{作品名}.md`

## 触发条件

### 主要触发模式

当用户表达中**同时满足**以下两个条件时触发：

**条件 1：包含作品标识**
- 书名号包裹的作品名：`《作品名》`
- 或明确的作品提及：`xxx 的 xxx`

**条件 2：包含台词相关关键词**
- "台词"、"名言"、"语录"、"经典句子"、"金句"
- "下载"、"整理"、"保存"、"收集"、"获取"、"想要"

### 触发示例

✅ **会触发的表达：**
- "下载《让子弹飞》的经典台词"
- "把周星驰的《大话西游》台词保存到本地"
- "整理一下《霸王别姬》的经典台词"
- "下载鲁迅《阿 Q 正传》的名言"
- "帮我收集《肖申克的救赎》里的经典句子"
- "想要《教父》里面的经典台词"
- "下载《千与千寻》的经典语录"

❌ **不会触发的表达：**
- "《让子弹飞》好看吗？"（无台词相关词）
- "姜文导演过哪些电影？"（无台词相关词）
- "今天天气不错"（完全无关）

### 信息提取规则

**提取作者/导演名：**
- 模式 1：`[作者/导演名] 的《作品名》` → 提取"作者/导演名"
 - 例："姜文的《让子弹飞》" → 作者=姜文
- 模式 2：`[动词][作者/导演名][作品名]` → 提取"作者/导演名"
 - 例："下载鲁迅《阿 Q 正传》" → 作者=鲁迅
- 模式 3：只有作品名 → 作者名留空，后续搜索获取

**提取作品名：**
- 优先提取书名号 `《》` 中的内容
- 其次提取引号中的内容
- 清理标点符号

### 情感分类体系

**五大情感分类：**
1. **励志/鼓舞**：积极向上、激励人心的台词（适合开工提醒、鼓舞士气）
   - 关键词：公平、正义、站着、奋斗、理想、信念、希望
   - 示例："我要站着，还把钱挣了！"、"我来鹅城只办三件事：公平！"

2. **爱情/温情**：表达爱意、亲情、友情的台词（适合情感类内容）
   - 关键词：爱、喜欢、幸福、等待、十年、缘分、一生、永远
   - 示例："曾经有一份真诚的爱情放在我面前"、"如果非要加上一个期限，我希望是一万年"

3. **幽默/讽刺**：搞笑、讽刺、黑色幽默的台词（适合轻松场景、缓解压力）
   - 关键词：翻译、惊喜、师爷、县长、扯蛋、荒唐、滑稽
   - 示例："翻译翻译，什么叫惊喜？"、"地主家也没有余粮啊"

4. **哲理/感悟**：蕴含人生哲理、深度思考的台词（适合深度对话、反思时刻）
   - 关键词：人生、意义、活着、命运、人性、现实、相信、选择
   - 示例："如果你活着，早晚都会死；如果你死了，你就永远活着"

5. **经典/脍炙人口**：广为流传、最具代表性的台词（通用场景，默认分类）
   - 特征：无法明确归类到其他四类的台词
   - 示例："让子弹飞一会儿"、"没有你，对我很重要"

**分类规则：**
- 使用关键词匹配 + 特殊规则双重机制
- 如果无法明确判断情感倾向，统一归入"经典/脍炙人口"
- 每条台词只归属一个主要分类
- 分类标签在输出中作为章节标题使用

## 输出规范

```
/root/data/{作者名}_{作品名}.md
```

**说明：**
- 如果有明确的作者（如小说改编），使用作者名
- 如果是电影，使用导演名替代作者名
- 作品名使用中文，去除标点符号
- 示例：
  - `/root/data/姜文_让子弹飞.md`
  - `/root/data/葛优_甲方乙方.md`
  - `/root/data/鲁迅_阿Q正传.md`

## 工作流程

1. **意图识别**：确认用户要求下载某部作品的台词
2. **信息提取**：
 - 作品名（电影名、书名）
 - 作者名/导演名
 - 其他相关信息（主演、年份等）
3. **网络搜索**：搜索该作品的经典台词
4. **内容整理**：按照标准格式整理台词
5. **情感分类**（关键步骤）：
 - 对每条台词进行情感分析，归入以下五类之一：
 - `励志/鼓舞`：积极向上、激励人心的台词
 - `爱情/温情`：表达爱意、亲情、友情的台词
 - `幽默/讽刺`：搞笑、讽刺、黑色幽默的台词
 - `哲理/感悟`：蕴含人生哲理、深度思考的台词
 - `经典/脍炙人口`：广为流传、最具代表性的台词（默认分类）
 - **分类规则**：如果无法明确判断情感倾向，统一归入"经典/脍炙人口"
 - 在输出中按分类组织台词结构
6. **文件保存**：保存到 `/root/data/{作者名}_{作品名}.md`，确保包含分类标签
7. **结果反馈**：告知用户保存位置、文件大小、分类统计信息
8. **连续任务处理**：如用户要求下载多部作品，按 `references/batch-download-protocol.md` 执行

## 示例

### 示例 1：电影台词（带情感分类）
**用户输入：**
> 把姜文的《让子弹飞》经典台词下载到本地

**处理流程：**
1. 识别意图：下载电影台词
2. 提取信息：导演=姜文，作品=让子弹飞
3. 搜索台词：搜索"让子弹飞 经典台词"
4. 内容整理：按照标准格式整理台词
5. **情感分类**：
 - 对每条台词进行情感分析，归入五类之一：
 - `励志/鼓舞`：如"我要站着，还把钱挣了！"
 - `幽默/讽刺`：如"翻译翻译，什么叫惊喜？"
 - `哲理/感悟`：如"人们不愿意相信一个土匪的名字叫牧之"
 - `经典/脍炙人口`：如"让子弹飞一会儿"
 - 无法明确判断的归入"经典/脍炙人口"
6. 保存文件：`/root/data/姜文_让子弹飞.md`，包含分类标签
7. 反馈结果：告知用户保存位置和分类统计

### 示例 2：文学作品
**用户输入：**
> 下载鲁迅《阿 Q 正传》的经典名言

**处理流程：**
1. 识别意图：下载文学作品名言
2. 提取信息：作者=鲁迅，作品=阿 Q 正传
3. 搜索内容：搜索"阿 Q 正传 经典语录"
4. 保存文件：`/root/data/鲁迅_阿 Q 正传.md`

### 示例 3：仅提供作品名
**用户输入：**
> 整理一下《霸王别姬》的经典台词

**处理流程：**
1. 识别意图：下载电影台词
2. 提取信息：作品=霸王别姬

### 示例 4：连续下载多部作品（批量任务）
**用户输入：**
> 帮我把周星驰的大话西游经典台词下载下来，成功之后，把阿 Q 正传也下载下来

**处理流程：**
1. 识别为连续任务：先处理《大话西游》，再处理《阿 Q 正传》
2. 第一步：
   - 提取信息：导演=周星驰，作品=大话西游
   - 搜索台词：搜索"大话西游 经典台词 周星驰 至尊宝 紫霞仙子"
   - 保存文件：`/root/data/周星驰_大话西游.md`
   - 反馈第一步完成
3. 第二步：
   - 提取信息：作者=鲁迅，作品=阿 Q 正传
   - 搜索内容：搜索"阿 Q 正传 鲁迅 经典名言"
   - 保存文件：`/root/data/鲁迅_阿 Q 正传.md`
   - 反馈第二步完成
4. 最终汇总：提供完整任务总结表格

**关键点：**
- 分步执行，每步完成后立即反馈
- 保持文件命名一致性（导演/作者_作品名.md）
- 最终提供汇总表格，包含文件名、大小、行数等信息
- 明确告知用户所有文件保存位置

## 工具依赖

- `web_search`: 搜索经典台词
- `web_extract`: 提取网页内容
- `write_file`: 保存文件到本地
- `execute_code` (可选): 处理文件名、格式化等

## 注意事项

1. **文件名规范**：
   - 去除特殊字符（如 `《》`、`""` 等）
   - 使用下划线连接作者和作品
   - 统一使用 `.md` 扩展名

2. **内容质量**：
   - 确保台词准确性
   - 优先选择权威来源（豆瓣、维基、官方资料）
   - 标注来源和可能的版本差异

3. **版权说明**：
   - 在文件末尾添加版权声明
   - 注明整理目的为个人学习使用

4. **目录检查**：
   - 确保 `/root/data/` 目录存在
   - 如不存在则创建

## 错误处理

- **找不到台词**：告知用户未找到相关台词，建议检查作品名
- **无法确定作者**：使用"未知作者"或导演名替代
- **保存失败**：检查目录权限，提示用户手动保存

## 扩展能力

- 支持批量下载（多部作品）
- 支持按主题分类（如"励志类"、"爱情类"）
- 支持导出为其他格式（JSON、TXT）
