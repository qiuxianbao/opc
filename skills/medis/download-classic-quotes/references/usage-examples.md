# Download Classic Quotes - 使用示例

## 技能说明

**技能名称**: `download-classic-quotes`

**功能**: 自动识别用户下载经典台词的意图，搜索并整理电影/文学作品的经典台词，保存到 `/root/data/` 目录。

---

## 触发条件

用户表达需**同时满足**：
1. 包含作品名（电影名、书名）
2. 包含台词相关词汇（"台词"、"名言"、"语录"、"经典句子"等）
3. 包含动作词汇（"下载"、"整理"、"保存"、"收集"等）

---

## 使用示例

### 示例 1：下载电影台词（带导演名）

**用户输入**:
```
下载姜文的《让子弹飞》经典台词
```

**执行流程**:
1. 识别意图：✅ 下载台词
2. 提取信息：导演=姜文，作品=让子弹飞
3. 搜索台词：搜索"让子弹飞 经典台词"
4. 保存文件：`/root/data/姜文_让子弹飞.md`
5. 返回结果

**输出**:
```
✅ 已为您下载《让子弹飞》的经典台词
保存位置：/root/data/姜文_让子弹飞.md
文件大小：3.9 KB
```

## 文件命名规则

```
/root/data/{作者名或导演名}_{作品名}.md
```

**说明**:
- 作者名优先于导演名
- 电影使用导演名
- 作品名去除标点符号
- 使用下划线连接

**示例**:
| 作品 | 文件名 |
|------|--------|
| 姜文《让子弹飞》 | `姜文_让子弹飞.md` |
| 葛优《甲方乙方》 | `葛优_甲方乙方.md` |
| 鲁迅《阿 Q 正传》 | `鲁迅_阿 Q 正传.md` |
| 陈凯歌《霸王别姬》 | `陈凯歌_霸王别姬.md` |

## 不会触发的情况

以下表达**不会**触发此技能：

❌ "《让子弹飞》好看吗？" - 无台词相关词
❌ "姜文导演过哪些电影？" - 无台词相关词
❌ "今天天气不错" - 完全无关
❌ "推荐几部经典电影" - 无具体作品名

## 技术实现

### 核心脚本
`/root/.hermes/skills/media/download-classic-quotes/scripts/download_quotes.py`

### 主要功能
1. `sanitize_filename()` - 清理文件名
2. `format_quotes_content()` - 格式化台词内容
3. `save_quotes_file()` - 保存文件
4. `extract_movie_info_from_search()` - 从搜索提取信息

## 扩展用法

### 批量下载
```python
works = [
    ("姜文", "让子弹飞"),
    ("周星驰", "大话西游"),
    ("陈凯歌", "霸王别姬"),
]

for author, title in works:
    download_quotes(author, title)
```

### 自定义输出目录
```python
save_quotes_file(author, title, content, base_dir="/custom/path")
```
